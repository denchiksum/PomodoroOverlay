# Pomodoro Overlay

A transparent, top-right anchored Pomodoro timer that stays on top of other windows

## Features
- Floating overlay with drag support
- Configurable durations and descriptions
- Random notification sound upon completion (Place your own in .\sounds)
- Local session logging (Markdown files in .\logs)
- Volume control via settings window

## Install
Download release for Windows: https://github.com/den4iksum/PomodoroOverlay/releases

## Tech Stack
- **Frontend:** React + TypeScript + Vite
- **Backend:** Rust + Tauri
- **Styling:** CSS (Glassmorphism)
